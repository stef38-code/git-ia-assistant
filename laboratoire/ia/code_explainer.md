# Assistant d'Explication de Code (Mentor IA)

## 💡 Concept
Permettre au développeur de demander une explication détaillée d'une partie du code, d'un fichier complet ou des changements introduits dans un diff. L'objectif est pédagogique : comprendre "pourquoi" et "comment" au lieu de simplement corriger.

## 🚀 Fonctionnalités
- **Explication ligne par ligne** : Analyser un bloc de code complexe et expliquer la logique métier et technique.
- **Analyse d'impact** : Expliquer les conséquences potentielles d'une modification de code sur le reste du système.
- **Support multilingue** : Pouvoir poser des questions sur le code dans une langue différente de celle des commentaires.
- **Optimisation de la lecture** : Générer des résumés de fichiers longs pour une prise en main rapide d'un nouveau projet.

## 🛠️ Intégration Git
- **`ia explain <file>`** : Explique le contenu global du fichier.
- **`ia explain --diff`** : Analyse les changements en cours et explique ce qu'ils font réellement (traduire le code en intentions).

## 📊 Valeur ajoutée
- Réduction de la courbe d'apprentissage sur les projets complexes.
- Amélioration de la compréhension technique au sein de l'équipe.
- Aide précieuse pour l'onboarding de nouveaux développeurs.
