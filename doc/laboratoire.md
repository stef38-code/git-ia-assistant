# Laboratoire - Idées à explorer

Le dossier `laboratoire/` contient des scénarios de fonctionnalités sous forme de documents Markdown. Certains scénarios sont déjà implémentés, d'autres sont en cours de développement ou à l'état d'idées.

## 🧪 Scénarios documentés

### 1. **Commit avec versioning automatique**
Automatise complètement le workflow de versioning lors d'un commit. Déjà implémenté via `git-ia-commit-version`.


### 3. **Intégration JIRA**
Synchroniser automatiquement Git avec JIRA (Auto-commenting, transitions, TODO → Tickets).

### 4. **Lint & Format Intelligent**
Détecte et corrige automatiquement les violations de linting avec explications IA.

### 5. **Coverage & Test Assistant**
Analyse la couverture de code et génère automatiquement les tests manquants.

### 6. **Assistant d'Explication de Code (Mentor IA)**
Explique le code ligne par ligne ou résume les intentions derrière un diff complexe pour faciliter la compréhension.

### 7. **Scénario : Journée type d'un développeur**
Un workflow complet intégrant toutes les fonctionnalités IA pour une productivité maximale (Expliquer -> Refacto -> Review -> Commit -> Release). Voir `laboratoire/scenarios/journee_type_developpeur.md` et sa version algorithmique `laboratoire/scenarios/journee_dev.md`.
